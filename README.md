# Hair App

Sistema de agendamentos para salão de beleza com interface moderna e responsiva.

## Sobre o Projeto

Aplicação web para gerenciamento de agendamentos de salão de beleza, desenvolvida com React e TypeScript. Permite selecionar data, horário e cadastrar clientes de forma intuitiva e visual.

## Funcionalidades

- Seleção de data com calendário interativo
- Escolha de horários disponíveis organizados por período (manhã, tarde, noite)
- Cadastro de nome do cliente
- Visualização de horários já agendados
- Interface responsiva para desktop, tablet e mobile

## Tecnologias Utilizadas

- **React 19.2.0** - Biblioteca JavaScript para construção de interfaces
- **TypeScript 5.9.3** - Supertipo do JavaScript com tipagem estática
- **Vite 7.2.4** - Build tool e servidor de desenvolvimento
- **React Day Picker 9.13.0** - Componente de calendário
- **date-fns 4.1.0** - Biblioteca para manipulação de datas

## Como Rodar

### Instalar Dependências

```bash
npm install
```

### Modo de Desenvolvimento

```bash
npm run dev
```
